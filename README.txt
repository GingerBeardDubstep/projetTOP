Pour lancer le programme principal rendez vous dans Scala IDE puis executer main.scala

Un affichage se lance dans la console.
Inscrivez la lettre "A" ou la lettre "B" pour acceder à la suite (toutes autres lettres, ou lettres non majuscule relance le début du programme)
Ensuite suivez les indications affichées à l'écran
Si vous avez choisi "B" :
suivez les nouvelles instructions qui vous font choisir entre "A", "B", ou "C"(toutes autres lettres, ou lettres non majuscule relance ce choix)
Dans les choix "B" et "C", il vous sera demandé de choisir entre "A" et "B" (toutes autres lettres, ou lettres non majuscule effectue "A" par défaut)
